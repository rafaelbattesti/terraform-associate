# Devcontainer

Creates the environment in a development container, and allows you to install features.

Today features are:

1. Terraform v1.4.5
2. terraform-docs version v0.16.0
3. node v18.16.0
4. go version go1.20.3
5. aws-cli/2.11.12
6. checkov 2.3.165

For more features see [features](ghcr.io/devcontainers/features).
